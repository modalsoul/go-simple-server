go-simple-server
============

go-simple-server is simple http servise using go-lang.

Usage
-----
    > cd src/main
    > go install
    > cd ../../bin
    
    > ./main

Default http port number is 3000.

You can use 

`http://localhost:3000/`

#### Configuration
##### Change port number
For example use 3001 port.

Use `-port` parameter

    > ./main -port 3001
